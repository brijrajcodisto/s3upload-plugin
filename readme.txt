== Description ==

**s3Uploads is a cloud storage and CDN delivery provider for your WordPress media library and other uploads. The s3Uploads plugin allows you to easily connect an unlimited number of sites to your s3Uploads cloud account for offloading your files, improving site performance, lowering hosting costs, and serving files faster to your visitors.**
